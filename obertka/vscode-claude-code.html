🧰 Как пользоваться VS Code и Claude Code

Рабочее место вайбкодера — настройка с нуля

Это инструмент, в котором ты будешь собирать свои AI-системы. Сначала — зачем именно он, потом пошаговая настройка.

1. Почему VS Code + Claude Code, а не «обычный ИИ-чат»

Обычный чат (на сайте) умеет только разговаривать: ты копируешь текст туда-сюда руками. Он не видит твои файлы, не помнит проект и ничего не делает сам.

VS Code + Claude Code — это рабочее место, где ИИ:
- видит всю папку проекта — читает твои документы, таблицы, презентации напрямую;
- сам создаёт и редактирует файлы, пишет код, запускает команды;
- помнит контекст проекта между сессиями (через файлы памяти);
- подключается к Telegram, твоим данным, внешним сервисам;
- умеет навыки (скиллы) — готовые экспертные модули.

Простыми словами: чат — это собеседник, а Claude Code — это исполнитель, который живёт внутри твоего проекта.

📌 На воркшопе «обёртку» мы собираем в обычном чате (так проще стартовать с нуля). VS Code + Claude Code — следующий уровень, для тех, кто пойдёт строить инструменты. Этот гайд — для него.

2. Пошаговая настройка

Шаг 1. Поставить VS Code
Скачай с code.visualstudio.com → установи (как обычное приложение). VS Code — это «редактор», твоё рабочее окно.

Шаг 2. Поставить расширение Claude Code
Внутри VS Code слева — иконка «Extensions» (квадратики). Найди Claude Code (от Anthropic) → Install. Слева появится иконка Claude — это панель, где ты с ним общаешься.

Шаг 3. Войти в аккаунт
Открой панель Claude → войди аккаунтом с подпиской Claude Max (тот же, что используешь). Это даёт доступ к сильной модели.

Шаг 4. Открыть папку проекта
File → Open Folder → выбери папку своего проекта (или создай новую). Claude Code будет работать именно с тем, что в этой папке.

Шаг 5. Запустить разговор
В панели Claude снизу — поле ввода. Пиши задачу человеческим языком. Он сам читает файлы, предлагает план, делает.

3. Контроль расходов (лимит «токенов»)

«Токены» — это единицы, которыми считается работа ИИ. Чтобы не было сюрпризов:
- На подписке Max лимит — это лимит твоего тарифа (отдельно платить за токены не нужно). Следи за остатком командой /usage в панели Claude.
- Держи контекст коротким — это и есть экономия: команда /clear (начать чистую сессию) и /compact (сжать историю), когда чат разросся.
- Если используешь API-биллинг (не подписку) — поставь лимит трат в консоли Anthropic (console.anthropic.com → Limits), чтобы был жёсткий потолок.
- Правило: одна задача — один сфокусированный чат. Не вали всё в один бесконечный диалог.

4. Какие скиллы подгрузить сразу

Скиллы — готовые экспертные модули, которые усиливают Claude под конкретные задачи. Полезный стартовый набор для бизнес-задач:
- Маркетинг / трафик / лидген — для офферов, воронок, контента.
- Конверсионный дизайн сайтов — для лендингов и страниц.
- Deep research — для глубокого ресёрча рынка/конкурентов.
- Генерация изображений — для обложек, артов, медиа к постам.
- Работа с Claude API — если будешь строить свои AI-инструменты.

Как добавить: скиллы ставятся как плагины (через маркетплейс) или кладутся в папку ~/.claude/skills/. На старте достаточно знать, что они есть — подключим под твою задачу. Claude сам подскажет, какой скилл применить.

5. Какие форматы файлов читает Claude Code

Можешь смело загружать в проект (просто положи файл в папку или укажи путь):

Тип Форматы Примеры что грузить
📄 Документы .pdf, .docx (Word), .txt, .md презентации, договоры, описания продуктов, транскрипты
📊 Таблицы .xlsx (Excel), .csv прайсы, юнит-экономика, базы, отчёты
🖼 Изображения .png, .jpg скриншоты, фото, схемы (он их «видит»)
🌐 По ссылке веб-страницы, Google-доки (по ссылке) сайт компании, статьи
💻 Код / данные любой код, .json, .ipynb если уже есть проект

Правило загрузки: сначала положи/прикрепи ВСЕ материалы, потом давай промпт. Чем богаче контекст — тем точнее результат. Чего нет — ИИ не выдумывает (помечает 🔴).
