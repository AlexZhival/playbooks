Установка с нуля — VS Code + Claude Code (для новичка, macOS)

Это самый простой путь: ставим VS Code, внутри него — расширение Claude Code, входим по подписке и запускаем. Терминал не нужен. Делай по шагам сверху вниз.

Что нужно заранее: подписка Claude Pro или Max (без неё войти не получится). Оформляется на сайте Claude, оплата картой.

Шаг 1 · Установить VS Code

  1. Открой сайт code.visualstudio.com.
  2. Нажми большую кнопку «Download for macOS» — скачается файл.
  3. Открой скачанный файл и перетащи «Visual Studio Code» в папку «Программы» (Applications).
  4. Запусти Visual Studio Code из «Программ».

Готово, когда: VS Code открылся (тёмное окно с приветствием).

Шаг 2 · Установить расширение Claude Code

  1. Слева в VS Code — вертикальный ряд иконок. Нажми «Extensions» (квадратик из кубиков) или Cmd + Shift + X.
  2. В строке поиска набери Claude Code.
  3. Выбери расширение Claude Code от Anthropic → нажми Install.
  4. Если VS Code попросит — перезапусти его (или Cmd + Shift + P → набери Reload Window → Enter).

Альтернатива: открой в браузере ссылку vscode:extension/anthropic.claude-code — она сама откроет VS Code на странице расширения.

Готово, когда: в расширениях у «Claude Code» появилась кнопка «Uninstall» / «Disable» (значит, установлено).

Шаг 3 · Войти по подписке (Pro/Max)

  1. Открой панель Claude Code: нажми значок Claude Code (искра ✱) — он в правом верхнем углу при открытом файле, либо «✱ Claude Code» в строке состояния внизу окна.
  2. Нажми Sign in — откроется браузер.
  3. Войди в свой аккаунт Claude (тот, где Pro или Max) и подтверди вход.
  4. Вернись в VS Code. Повторно входить больше не нужно.

Готово, когда: в панели Claude Code появилось поле для ввода (чат готов принять команду).

Шаг 4 · Открыть папку проекта

  1. Файл → Открыть папку…
  2. Выбери распакованную папку «Обёртка — стартовый набор (практикум)» (из стартового набора — см. домашнее задание).
  3. Слева появится дерево файлов этой папки.

Готово, когда: в дереве слева видно CLAUDE.md, папки конвейер/, материалы-компании/, обёртка/.

Шаг 5 · Первая команда

  1. Сначала положи свои исходники компании в папку материалы-компании/ (см. ДЗ).
  2. В панели Claude Code напиши и отправь:
Прочти CLAUDE.md и начни вести меня по сборке обёртки
  1. Дальше просто отвечай на его вопросы. Готовые артефакты будут появляться в папке обёртка/.

Готово, когда: Claude поприветствовал, прочитал материалы и задал первый вопрос.

⚙️ Рекомендуемые настройки (выставь один раз)

Открой меню режимов в панели Claude Code (там, где Modes и Effort — вызывается Shift + Tab или из меню). Выставь так:

  • Режим (Modes): «Edit automatically». Claude сам сохраняет артефакты в папку, не спрашивая разрешение на каждый файл, — а потенциально опасные команды всё равно спросит. Для работы в своей папке (🟢 зелёная зона) это удобно и безопасно; любую правку можно откатить.
  • ⚠️ «Bypass permissions» — НЕ включай. Это режим без подтверждений даже на опасные действия — он только для изолированных машин/красной зоны.
  • Хочешь видеть каждый шаг — выбери «Ask before edits» (спросит перед каждой правкой) или «Plan mode» (сначала покажет план, потом сделает).
  • Effort: High. Больше «думанья» = глубже и качественнее стратегические тексты. Для обёртки — High.
  • Модель: для глубины выбери Opus (или «лучшую доступную»). Переключение — команда /model.
  • Скиллы и плагины ставить НЕ нужно. Для сборки обёртки всё уже задано в CLAUDE.md — не усложняй.

Длинная сессия (6–7 артефактов подряд): если чат «тяжелеет» — команда /context покажет, что занимает место, а /compact сожмёт историю, сохранив главное. Файл CLAUDE.md подгружается всегда — инструкции оттуда не теряются.

Частые вопросы

Нужен ли терминал? Для сборки обёртки — нет. Панель Claude Code в VS Code = весь интерфейс. Терминал нужен только для отдельных команд (например, подключение Telegram-бота), и это встроенный терминал VS Code (View → Terminal или Ctrl + `) — отдельное приложение Terminal не требуется.

Нужен ли Bun? Нет. Расширение несёт всё необходимое в себе.

Не вижу значок Claude Code. Перезапусти VS Code: Cmd + Shift + P → Reload Window → Enter. Убедись, что расширение установлено (Шаг 2).

Не пускает войти / просит оплату. Проверь, что у тебя активна подписка Claude Pro или Max на том аккаунте, которым входишь.

Хочу терминальный Claude Code и Telegram-бота. Это следующий уровень — см. памятку Настройка окружения (Claude Code + бот). Для обёртки он не нужен.
