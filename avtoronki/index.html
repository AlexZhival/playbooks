VPC — Value Proposition Canvas

Карта ценности
Создатели выгодкак даём результат
Продукты и услугичто предлагаем
Обезболивающиекак снимаем боли
⟷
FIT
Профиль клиента
Выгодыкакой результат хочет
Задачи (Jobs)что пытается сделать
Боличто мешает / страшит
Правила заполнения
  • Сначала правая сторона (профиль клиента), потом левая — не наоборот
  • Дословный язык клиента — из звонков, отзывов, переписок, не свои формулировки
  • Ранжируй по важности — топ-боли и топ-выгоды сверху
  • FIT = каждое «обезболивающее» бьёт в конкретную боль, каждый «создатель выгод» — в конкретную выгоду
Оффер = точное совпадение сторон. Промпт сборки: Промпты/0-vpc.md

Вечный догрев

⚡

1-дневный

6–12 сторис. Концепция «смотри и повторяй». Быстрый прогрев к конвертору.

📅

5-дневный

По дню на ступень: доверие → мифы → продукт → результаты → дефицит. Можно сжать до 1–2–3 дней.

🎬

2-недельный реалити

4 этапа: доверие → этапы ниши → бесплатник → дожим. Шоу + большая идея.

Прогрев — это вызов жгучего желания получить точку Б, к которой ведёт продукт. Полная структура, тайминги по дням и плейбук — по кнопкам выше.
